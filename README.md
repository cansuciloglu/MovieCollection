# Movie Collection

# Technologies Used
- Spring Boot
- Jpa
- Maven
- Thymeleaf
- Bootstrap


# SETUP
* Create database named movie_collection. Import the movie_collection.sql file. 
* Import Maven Project and run with Java Application

# Functionalities

- ADD MOVIE <br>
- VIEW MOVIE <br>
- SEARCH MOVIE <br>
- SORT BY YEAR MOVIE <br>
- EDIT MOVIE <br>
- DELETE MOVIE <br>
